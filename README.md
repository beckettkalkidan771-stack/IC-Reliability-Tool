# 集成电路封装可靠性加速模型计算器

## 1. 项目背景
[cite_start]本工具是《集成电路封装技术与工程应用》数字教材的配套交互资源 [cite: 272]。
[cite_start]对应章节：**第三篇 第8章 可靠性验证工程与失效分析** [cite: 154]。
[cite_start]适用任务：**学习任务 8.1 可靠性测试方案设计** [cite: 156]。

## 2. 核心功能
* [cite_start]**Arrhenius 模型计算**：支持高温工作寿命（HTOL）加速因子计算 [cite: 159]。
* [cite_start]**Peck 模型计算**：支持高加速应力测试（HAST）和温湿度偏置（THB）加速因子计算 [cite: 160]。
* **工程判定**：内置寿命预测逻辑，自动判定方案是否满足车规级/工业级寿命目标。
* [cite_start]**标准参考**：内置 JEDEC 相关测试标准参数对比表 [cite: 160]。

## 3. 技术参数说明
* **编程语言**：Python 3.x
* **核心库**：Streamlit, Numpy
* **物理参数**：默认激活能 $E_a = 0.7eV$，湿度指数 $n = 3.0$。

## 4. 给出版社的部署建议
本程序基于 Streamlit 框架开发。
- **在线嵌入**：建议部署于 Streamlit Community Cloud 或企业级容器云，通过 `<iframe>` 嵌入教材 HTML 页面。
- [cite_start]**离线访问**：可生成对应 URL 的二维码，印制在教材任务 8.1 标题旁 [cite: 161]。

## 5. 开发建议与反馈
[cite_start]本工具旨在培养职业本科学生的工程思维，支持学生在实验任务中进行“数据驱动”的方案设计 [cite: 161]。
